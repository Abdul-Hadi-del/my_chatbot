My Chatbot 🤖
A simple AI chatbot built with Python and Groq API.
What it does
Chat with an AI directly from your terminal. Ask anything and get instant responses.
Tech Stack

Python 3.14
Groq API (LLaMA 3.3 70B model)

How to run

Clone the repo
Install dependencies:

pip install groq

Add your Groq API key in chatbot.py
Run:

python chatbot.py
